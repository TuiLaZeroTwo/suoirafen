<h2>how to use?</h2>

<p>1. install node</p>

```
nodejs.org
```

<p>2. clone this repo</p>

```
$ git clone https://github.com/rylzsense/nefarious.git
```

<p>3. cd to repo</p>

```
$ cd nefarious
```

<p>4. install libs</p>

```
$ npm i
```

<p>5. add your credentials in configuration.json</p>

<p>6. run</p>

```
$ node .
```
